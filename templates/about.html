關於 About HANCOC

HANCOC終於在網站上上架了！跟之前的程式大不相同的是，這次不需要下載任何東西就能讓這個工具運作了。以下就是操作的步驟：

  1. 點進去HANCOC的首頁。
  2. 把MuseScore的樂譜檔匯出成未壓縮的MusicXML，並把副檔名設為「.xml」。
  3. 從你的電腦中選取要加上和弦記號的MusicXML檔案。
  4. 按「下載樂譜」。
  5. 瀏覽器會顯示加上了和弦記號的MusicXML檔案。
  6. 點右鍵並選擇「儲存為...」。
  7. 在儲存檔案的時候，記得把副檔名也設為「.xml」。
  8. 大功告成！
The HANCOC tool is finally up on the internet! Unlike the previous program, you won't have to download any software. Here's how it works:

  1. Go to HANCOC's home page.
  2. Export your MuseScore file to uncompressed MusicXML format, and set the file's filename extension to ".xml".
  3. Find the MusicXML file you are uploading.
  4. Click "Download score".
  5. The browser will show the MusicXML file with chord notations added.
  6. Right-click and choose "save as...".
  7. When saving the file, make sure to set the file's filename extension to ".xml".
  8. Well done!
